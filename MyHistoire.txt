<h1 id="mon-histoire">Mon Histoire</h1>
<p><em><strong>J’ai d’abord découvert <a href="http://minecraft.net">Minecraft</a>, sur Xbox360, mais pas son côté Pvp, craft… Mais le côté Créatif: surtout la redstone. Au fil du temps, j’ai découvert Scratch, puis Minecraft : Bedrock Edition, la copie Minecraft de Microsoft. Puis, début 2021, j’ai eu Minecraft : java Edition, où j’ai découvert un “Secret” dans le launcher : Les versions en Json. J’ai ensuite découvert <a href="http://eclipse.org">Eclipse</a>, le célèbre logiciel de programmation Java. Grâce à lui, j’ai crée un Créateur de Versions de Minecraft, compatible avec les mods. Java, je l’ai commencé à l’appendre fin Mars 2021. J’ai ensuite découvert <a href="http://mcreator.net">MCreator</a>, le logiciel(de plus en plus connu) de création de mods minecraft. Plus tard, début juillet 2021, j’ai appris le HTML et le JavaScript, qui me servent pour les github pages et FirePage. Ensuite, le 15/09/21, j’ai appris le <a href="http://php.net">Php</a>. Le 21/11/22, j’ai appris l’arduino, et nous voilà à aujourd’hui</strong>
